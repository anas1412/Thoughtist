**Thoughtist** is a premium, single-file spatial productivity tool that bridges the gap between chaotic brainstorming and structured execution. It treats your ideas as physical objects in an infinite workspace, allowing you to organize thoughts organically before morphing them into a structured Kanban workflow.

## ‚ú® Core Innovations

### 1. Kinetic Information Architecture
Unlike static lists, Thoughtist uses a custom **Coulomb Physics Engine**. 
- **Universal Repulsion:** Thoughts naturally push each other away to maintain clarity.
- **Gravitational Tags:** Nodes sharing the same tags exert a "spring force," pulling related ideas into organic clusters.
- **Viscous Movement:** Physics are tuned to feel "liquid," allowing nodes to drift satisfyingly into place.

### 2. Dual-View Morphing
Seamlessly switch between two mental states:
- **Spatial Mode:** An infinite, zoomable canvas for non-linear mind mapping.
- **Kanban Mode:** A structured board (Todo/Doing/Done) that automatically stacks nodes vertically based on their dynamic content height.

### 3. Infinite Navigation
- **Pan:** Middle Mouse Button (or Alt + Click) to drag the world.
- **Zoom:** Mouse Wheel to dive into details or see the big picture.
- **Smart Stacking:** Kanban columns automatically calculate real-time node heights to prevent overlaps.
